Description

This is a project i started because I am currently learning HTML (HyperText Markup Language), the first step on the way to learning front end web development.

I will continue by adding CSS, JavaScript and React as I learn more.

A side benefit could in time be to share the work I did while traing to become a helicopter Certified Flight Instructor, or CFI for short.

Initially my plan is to create a website that lists out categories of content or a header if you will, for everything I plan to share.

My long term goal is to gradually fill in content from my CFI binder, handouts and other knowledge i have gained through my flight training and working as a flight instructor.

Then I plan on integrating this content using CSS and JavaScript as I learn those languages.

CFI Binder

This chaptered document will be the main body of this website. It contains all the knowledge an airman or airwoman (come on, it's time) is required to know in order to legally act as a CFI inside the FAA regulations that govern aviation in the USA.

Articles

This section will contain an article written by me, that helped me learn a subject that was really confusing to me during training, namely the required awareness tring for the robinson helicopters.
